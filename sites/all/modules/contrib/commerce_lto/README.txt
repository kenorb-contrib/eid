
-- SUMMARY --

This module provides functionality to deal with date / time dependent prices.

It provides these features:
 - Define prices for specified date ranges.
 - Define prices for time units e.g. day / week
 - Price calculation based on the above mentioned features via a default rule.


-- REQUIREMENTS --

Commerce: http://drupal.org/project/commerce
Date: http://drupal.org/project/date


-- INSTALLATION --

* Install as usual, see http://drupal.org/documentation/install/modules-themes/modules-7 for further information.

* Make initial configuration: admin/commerce/config/lto

* Configure the newly added date field on the line item to fit to your needs.
